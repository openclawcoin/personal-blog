之前在网上看到一篇整理 OpenClaw Skills 的文章，写得很实用，就顺手拿来改写了一下，结合我自己的使用心得，供大家参考。

---

## 在装技能之前，第一步一定要先装 Skill-Vetter

它会帮你分析技能的安全风险，检查权限范围，发现红旗（red flag）和可疑模式。**在安装任何技能前先跑一遍，确保安全。**

---

## 第二个：Agent Browser（浏览器自动化之王）

浏览器是很多功能的基础。如果你想让 AI 真正帮你操作电脑、自动化网页任务，Agent Browser 是必须装的基础插件。

---

## 第三个：Tavily Web Search（AI 专属联网）

Tavily 是专为 AI Agent 设计的联网搜索工具。它比普通搜索引擎更懂 AI 的需求，可以返回结构化的搜索结果，支持多来源聚合，非常适合需要实时信息的场景。

---

## 第四个：Find-Skills（技能发现器）

帮你发现更多 Skills，持续扩展 AI 的能力边界。

---

## 第五个：Weather（天气信息）

简单实用，随时获取天气数据。

---

## 第六个：Self-Improving-Agent（自我进化）

让龙虾越用越聪明——记录错误、总结经验、持续优化。

---

## 第七个：Summarize（内容总结）

信息处理必备，一键总结文章、视频、本地文件。

---

## 第八个：Proactive Agent（主动提醒）

让龙虾变主动，不再被动等待指令。

---

## 第九个：Humanizer（人性化表达）

输出更自然，说人话、不生硬。

---

## 一键安装命令

```bash
openclaw plugins add skill-Vetter
openclaw plugins add agent-browser
openclaw plugins add tavily-search
openclaw plugins add find-skills
openclaw plugins add weather
openclaw plugins add self-improving-agent
openclaw plugins add summarize
openclaw plugins add proactive-agent
openclaw plugins add humanizer
```

## 安装顺序建议

| 顺序 | Skill | 为什么这个顺序 |
|------|-------|--------------|
| 1 | Skill-Vetter | 先装安全，再装别的 |
| 2 | Agent Browser | 浏览器是很多功能的基础 |
| 3 | Tavily | 让龙虾能联网搜索 |
| 4 | Find-Skills | 帮你发现更多技能 |
| 5 | Weather | 简单实用 |
| 6 | Self-Improving | 让龙虾越用越聪明 |
| 7 | Summarize | 信息处理必备 |
| 8 | Proactive | 让龙虾变主动 |
| 9 | Humanizer | 输出更自然 |
